A propos

La recherche sur la carte d'openstreetmap.org utilise nominatim et pour la France, la base de données est beaucoup moins conséquente que celle de la BAN. Cette petite application fait donc le lien entre la BAN et la carte d'openstreetmap. Vous lui donnez une adresse, elle vous donne une carte pour chaque adresse trouvée, le tout classé par pertinence :)

Cette application flask a été développée par Daniel Jakots.

Le code est disponible sur github. {% endblock %}
